# epic-chip-react-native

An elegant react native chip module.

## Installation

```sh
npm install epic-chip-react-native
```

## Usage

```js
import { EpicChip } from "epic-chip-react-native";

// ...

<EpicChip
  labelOptions={{
    label: "Epic Chip",
  }}
/>;
```
