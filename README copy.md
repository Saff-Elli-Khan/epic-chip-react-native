# epic-chip-react-native

An elegant react native chip module.

## Installation

```sh
npm install epic-chip-react-native
```

## Usage

```js
import EpicChipReactNative from "epic-chip-react-native";

// ...

const result = await EpicChipReactNative.multiply(3, 7);
```
